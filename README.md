# Kinopark

**Kinopark** is a web application for booking movie tickets, viewing movie schedules, and managing user accounts. The project is built using **Django**, providing a fast and convenient way for users to interact with the cinema services.

## Features

- User registration and authentication
- Viewing a list of movies and their schedules
- Booking tickets for selected movies
- User profile management
- Admin panel for managing movies and showtimes

## Technologies

- **Backend:** Django, Django REST Framework  
- **Frontend:** HTML, CSS, JavaScript (Bootstrap optional)  
- **Database:** SQLite (default, can be configured for PostgreSQL)  
- **Version Control:** Git
